HealAI Project Website is an AI-powered healthcare diagnostics platform designed to simplify and speed up medical report analysis. The system allows users to upload medical reports in PDF format and instantly receive easy-to-understand insights, risk indicators, and health summaries. It uses intelligent document processing and AI-driven analysis to detect abnormalities, highlight critical values, and provide personalized health recommendations. The platform is built to assist patients, doctors, and caregivers by reducing the time required to interpret complex medical data. HealAI focuses on accessibility, fast diagnostics, and user-friendly visualization, making healthcare information more understandable and actionable for everyone.


# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/31bb3083-9f3a-4c44-9a78-3c1834c03482 

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
